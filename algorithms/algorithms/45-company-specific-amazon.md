# Company Specific: Amazon

## Overview
Amazon focuses heavily on **Leadership Principles (LPs)** and **Object-Oriented Design** combined with DSA.

## Top Patterns for Amazon
1.  **BFS/DFS (Graphs/Trees)**: Amazon loves graph problems (e.g., delivery routes, package dependencies).
2.  **Sliding Window**: String manipulation.
3.  **Arrays/HashMaps**: Optimization problems.

## Top Questions (Last 6 Months)
1.  **Number of Islands**
2.  **Partition Labels**
3.  **Two Sum**
4.  **Reorder Data in Log Files**
5.  **LRU Cache**
6.  **Merge Intervals**
7.  **Search Suggestions System** (Trie)
8.  **Course Schedule**
9.  **Word Ladder**
10. **Critical Connections in a Network** (Tarjan's Algo - Hard)

## Interview Structure
*   **Online Assessment (OA)**: 2 coding questions (90 mins) + Work Style Assessment.
*   **Phone Screen**: 1 coding question.
*   **Onsite (Loop)**: 4-5 rounds.
    *   Each round has 15-20 mins of LPs.
    *   **Bar Raiser**: Focuses on LPs and probing depth.

## Strategy
*   **LPs are Key**: Prepare stories for "Customer Obsession", "Bias for Action", "Dive Deep".
*   **Clarify**: Amazon interviewers often give vague requirements to see if you ask questions.
