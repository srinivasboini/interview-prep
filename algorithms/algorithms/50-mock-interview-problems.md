# Mock Interview Problems

## Set 1: Arrays & Strings (45 mins)
1.  **Problem**: Longest Substring Without Repeating Characters.
2.  **Follow-up**: What if the set of characters is huge (Unicode)?

## Set 2: Trees & DFS (45 mins)
1.  **Problem**: Lowest Common Ancestor of a Binary Tree.
2.  **Follow-up**: What if nodes have parent pointers?

## Set 3: DP & Optimization (45 mins)
1.  **Problem**: Coin Change.
2.  **Follow-up**: What if we need to return the actual coins used?

## Set 4: System Design Mix (45 mins)
1.  **Problem**: Design Hit Counter.
2.  **Follow-up**: Scale to distributed system.

## Evaluation Criteria
*   **Code Quality**: Variable names, modularity.
*   **Correctness**: Passes test cases.
*   **Communication**: Clarity of thought.
*   **Optimization**: Did they find the optimal solution?
