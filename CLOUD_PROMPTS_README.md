## Usage

Use these prompts to create comprehensive interview preparation guides that will help senior engineers prepare for Staff/Principal Engineer level technical interviews at top financial institutions like JP Morgan, UBS, Goldman Sachs, etc.

Each prompt specifies:
- Content requirements and topics to cover
- Format requirements for each section
- Research requirements and authoritative sources
- Special requirements (diagrams, interview focus, enterprise context)
- Structure and organization options
- Quality standards
- Expected length and success criteria
